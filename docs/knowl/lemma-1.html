Lemma 1.7.

Let \(n \ge 1\)

  1. \begin{equation*} \psi \colon E(K)/ nE(K) \to \{ K \subseteq F \subseteq \overline K\} \end{equation*}
    \begin{equation*} P\mapsto K(\frac 1n P, E[n]) \end{equation*}
    is well defined.
  2. \(K(\frac 1n P, E\lb n \rb ) / K\) only ramifies at \(\ideal p | n \Delta_E\text{.}\)
  3. \begin{equation*} \Gal{K(\frac 1n P, E [n] )}{K} \le \ZZ/n \times \ZZ/n \end{equation*}
  4. There are only finitely many fields satisfying 2. and 3. so \(\image \psi\) is finite.
